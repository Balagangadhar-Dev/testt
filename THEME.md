## Usage Tips

1. **Consistency**: All interactive elements use consistent green shades
2. **Hierarchy**: Darker greens for backgrounds, lighter for accents
3. **Feedback**: Glows and shadows provide visual feedback
4. **Performance**: Animations are GPU-accelerated where possible

## Accessibility

- High contrast between text and backgrounds
- Green colors chosen for visibility
- Animations can be reduced via `prefers-reduced-motion`

**Design Philosophy**: Nature-inspired, modern, and energetic green theme that creates a calm yet engaging learning environment.
